#### Stack:
---
- **Front-end:** 
    - Typescript
    - React
    - Redux-toolkit
    - SCSS
    - Axios

- **Back-end:**
    - Typescript
    - NodeJS
    - Express
    - MongoDB / Mongoose
    
#### Installation:
---
**Client**
```
cd client
npm install
npm run dev
```

**Server**
```
cd server
npm install
npm run dev
```

You can use your own MongoDB database by simply changing the `DB_URL` connection link in the **.env** file in the **server** folder.
